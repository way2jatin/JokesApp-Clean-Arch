# Jokes

A simple Android app which displays jokes using API.


## Scenario

This app:
- Displays a list of jokes.


## Architecture

This app implemented using Clean Architecture containing 3 main layers:
1.  Presentation (used MVVM pattern for this layer)
2.  Data
3.  Domain


## Dependencies

- Retrofit
- Coroutines
- Moshi
- Room
- LiveData
- ViewModel
- Koin
- Mockk
- Jetpack Compose
